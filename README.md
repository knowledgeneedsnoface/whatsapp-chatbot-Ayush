# WhatsApp Chatbot

Real-time WhatsApp chatbot using WhatsApp Business API.

## Setup

1. Install dependencies:
```bash
   npm install
```

2. Copy `.env.example` to `.env` and add your tokens

3. Run the server:
```bash
   npm start
```

## Environment Variables

- `VERIFY_TOKEN` - Your webhook verification token
- `WHATSAPP_TOKEN` - WhatsApp API access token
- `PHONE_NUMBER_ID` - Your WhatsApp Phone Number ID
